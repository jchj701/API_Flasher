# API_Flasher
Word中的API命令格式刷

- 首个vba版本，已支持基本功能的格式刷
- 使用前需支持确保格式符合，有：表格后方没有额外的换行
- 使用前需支持确保格式符合，有：示例命令前缀包含“COMMAND>”前缀
- 使用时需要对待格式化内容进行选中
- 从第一个【命令】字段开始，前缀和标题暂不支持，有空再开发 
- 选中后，调用宏 API_IPMI_RunStyleFormat  即可
